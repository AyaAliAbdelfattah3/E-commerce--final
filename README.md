# 🛒 AyaShop – Modern E-Commerce Platform

**AyaShop** is a high-performance, fully responsive e-commerce application designed to provide a seamless online shopping experience. The project focuses on clean UI/UX, efficient state management, and real-time user interactions.

---

## ✨ Key Features

* **Dynamic Product Management:** Browse, filter, and view detailed product information.
* **Advanced Shopping Cart:** Full CRUD functionality (Add, Update, Remove) powered by **Redux Toolkit**.
* **Seamless Checkout Flow:** Managed via **Formik** for robust form handling and validation.
* **Interactive UI:** Smooth sliders, loading states, and modern toast notifications.
* **Contact System:** Integrated **EmailJS** to handle customer inquiries directly from the frontend.
* **Fully Responsive:** Optimized for all screen sizes using **Tailwind CSS**.

---

## 🛠️ Tech Stack

| Category | Technologies |
| :--- | :--- |

| **UI Components** | Swiper.js, SweetAlert2, React Hot Toast, React Spinners |
